## Overview

JoHex AI enables Python scripts running inside JoHex to communicate with modern AI models using the OpenAI-compatible Chat Completions API.

Before using any AI-powered features, you must configure an AI provider.

JoHex AI does **not** provide an AI model or API key. You may use any OpenAI-compatible AI service.

Supported providers include (but are not limited to):

- OpenAI
- Google Gemini
- NVIDIA Build
- DeepSeek
- OpenRouter
- Ollama
- LM Studio
- LocalAI
- vLLM
- Self-hosted OpenAI-compatible services

---

# Open AI Configuration

Open the following menu inside JoHex:

```text
Options
    └── Preferences
            └── AIConfig
```

Configure the following options.

---

## Endpoint URL

Specify the Chat Completions endpoint of your AI provider.

Examples:

```text
https://api.openai.com/v1/chat/completions
```

```text
https://integrate.api.nvidia.com/v1/chat/completions
```

```text
http://localhost:11434/v1/chat/completions
```

The endpoint must support streaming responses.

---

## API Key

Enter the API key provided by your AI service.

Examples:

```text
sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

or

```text
nvapi-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

The API key is stored locally and is only used when communicating with the configured AI provider.

---

## Model

Specify the model name.

Examples:

```text
gpt-4.1
```

```text
meta/llama-3.3-70b-instruct
```

```text
gemini-2.5-pro
```

```text
deepseek-chat
```

```text
qwen3-235b
```

The available model names depend on your selected AI provider.

---

## Maximum Response Tokens

Specifies the maximum number of tokens generated by the AI model.

Recommended values:

| Scenario | Recommended |
|----------|------------:|
| Quick explanation | 512 |
| Binary analysis | 1024 |
| Reverse engineering | 2048 |
| Detailed report | 4096 |

Higher values may increase response time and API usage.

---

## Maximum Upload Size

Specifies the maximum amount of extracted text or hexadecimal data sent to the AI model.

Recommended values:

| File Size | Upload Size |
|-----------|------------:|
| Small | 4096 characters |
| Medium | 8192 characters |
| Large | 16384 characters |

Larger uploads provide more context but consume more tokens.

---

## Temperature

Controls response randomness.

Recommended value:

```text
0.1
```

Lower values produce more deterministic technical analysis.

---

## Request Timeout

Maximum waiting time for AI responses.

Recommended value:

```text
60 seconds
```

---

## Streaming Response

Enable streaming mode.

Recommended:

```text
Enabled
```

Streaming allows JoHex to display AI responses in real time while they are being generated.

# Security

Your API key is stored locally on your computer.

JoHex only sends requests to the AI provider that you configure.

No binary data or file contents are uploaded unless you explicitly execute an AI command.

# Troubleshooting

## AI Connection Error

Please verify:

- Endpoint URL
- API Key
- Internet connection
- Firewall settings
- AI provider availability

---

## Authentication Failed

Please verify:

- Your API key is valid.
- Your API key has not expired.
- Your account has permission to access the selected model.

---

## Model Not Found

Please verify that the configured model name exactly matches one provided by your AI service.

---

## Request Timeout

Possible causes include:

- Slow network connection
- Large upload size
- AI provider overload

Reducing the Maximum Upload Size may improve response speed.

---

# Notes

JoHex AI is provider-independent.

Any AI service implementing the OpenAI Chat Completions API with streaming support can be integrated without modifying JoHex itself.
